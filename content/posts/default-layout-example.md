# 默认布局

这是默认的文章布局（不指定 `layout` 字段或使用 `layout: default`）。

## 特点

- 封面图片固定高度（176px）
- 标题和日期显示在图片下方
- 卡片内容清晰分离
- 适合大多数文章类型

## 使用方法

在文章的 frontmatter 中可以：

1. 不添加 `layout` 字段（会使用默认布局）
2. 或明确指定：

```yaml
layout: default
```
